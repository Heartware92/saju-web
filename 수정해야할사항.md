## 📊 만세력 페이지

### 1. 합·충·형·파·해 표기 — 반합/반방합 통합 (🟡)
**현상**:
- 우리 화면: "사오 **반방합** 화국", "신유 **반방합** 금국", "유사 **반합** 금국"
- 다른 어플들: 모두 "**반합**" 으로 통합 표기

**사용자 의견**: "다른 어플들 다 반합으로만 표기. 우리만 유난스러움"

**수정안 (옵션 🅐)**:
- 라벨/타이틀은 "반합" 으로 통합 표기 (다른 어플과 친숙도 맞춤)
- 부가 설명에 출처 명시 — 예: "사오 반합 화국 (사오미 방합 일부)" / "유사 반합 금국 (사유축 삼합 일부)"
- 명리 정확도 잃지 않으면서 사용자 친숙도 ↑

**위치**: `src/utils/sajuCalculator.ts` 의 `calculateInteractions()` 안 description 텍스트
- 1149행 근처 — 방합 description: `${matchVals.join('')} ${uniqueVals.size === 3 ? '방합' : '반방합'} ${element}국 - 같은 방향 ${element} 기운 결집`
- → `${matchVals.join('')} ${uniqueVals.size === 3 ? '방합' : '반합'} ${element}국 (${b1}${b2}${b3} 방합 ${uniqueVals.size === 3 ? '완성' : '일부'})`

**유지할 것**:
- 같은 합 1번만 표기 (다른 어플의 중복 카운트는 가독성 X)
- 8각형 시각화 그대로

### 4. 대운·세운·월운 — 진입 시 현재 시점 자동 펼침 (🟡)
**현상**:
- 페이지 진입 시 대운 카드 나열만 됨
- 사용자가 대운 카드 클릭해야 → 그 대운 범위 세운 강조
- 다시 세운 카드 클릭해야 → 그 해 월운 12개월 펼침
- **클릭 안 해도 자동으로 현재 대운 + 현재 연도 세운 + 현재 월운까지 다 펼쳐져 있어야 함**

**원인**:
- `SajuReport.tsx:1033, 1035` —
  ```ts
  const [selectedDwAge, setSelectedDwAge] = useState<number | null>(null);
  const [selectedYear, setSelectedYear] = useState<number | null>(null);
  ```
  초기값이 null → 클릭 전엔 펼침 안 됨

**수정안**:
- `selectedDwAge` 초기값을 **현재 나이가 속한 대운 시작 나이** 로 설정
  - 계산: `daeWoon` 배열에서 `currentAge >= startAge && currentAge < startAge+10` 인 대운 찾아 그 startAge
- `selectedYear` 초기값을 **현재 연도(`currentYear`)** 로 설정
- 월운은 `selectedYear` 만 채워지면 자동으로 펼쳐짐 (`monthlyFlow` useMemo 가 자동 계산)

**기대 동작**:
1. 페이지 진입 즉시 현재 대운 카드 강조 + 그 범위 세운 영역 자동 강조
2. 현재 연도 세운 카드 강조
3. 현재 연도의 12개월 월운 그리드 자동 노출
4. 사용자 클릭은 "다른 시기 보고싶을 때" 만 사용

### 6. 대운 표시 순서 — 우→좌 (전통) vs 좌→우 (현재) (🟢)
**현상**:
- 우리 화면: **8세 → 18세 → 28세** (좌→우, 서양식)
- 다른 어플: **28세 → 18세 → 8세** (우→좌, 전통식)

**원인**: `SajuReport.tsx:1095` `daeWoon.slice(0, 10).map((dw, idx))` — 단순 idx 순서. CSS flex 가 좌→우 방향.

**전통 명리**:
- 한자 세로쓰기 책 관습 → 우측이 시작·뿌리, 좌측이 미래
- 사주 본자체도 전통식: 시주(우) ─ 일주 ─ 월주 ─ 년주(좌) 인 어플 많음
- 우리는 사주도 시·일·월·년 (좌→우) 으로 표기 중 → 일관성 있음

**판단**:
- 좌→우 유지가 모바일·웹 UI에 자연스럽고 우리 사주 본자체와 일관
- 단, 정통 사주 사용자는 어색할 수 있음

**대안**:
- A) 그대로 유지 (현재 — 좌→우)
- B) 사주 본자체 + 대운·세운·월운 모두 우→좌 전환 (전통 통일)
- C) 사용자 토글 (불필요한 복잡성)

**사용자 결정 대기 — 일단 그대로 유지 권장**
