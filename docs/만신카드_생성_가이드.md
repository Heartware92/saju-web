# 만신 카드 일러스트 생성 가이드 (미드저니)

> 만신 타로 60장(신령 36 + 풍습 18 + 엽전 6) 일러스트 양산을 위한 확정 레시피.
> 2026-07-10 엽전패 6장 제작 세션에서 검증된 방법과 **실패 사례 전수 기록**.
> 새 카드를 뽑기 전에 반드시 이 문서를 먼저 읽는다.

## 1. 확정 표준 레시피 (이대로만 하면 됨)

| 항목 | 값 | 비고 |
|---|---|---|
| 모델 | **V7** | 옴니 레퍼런스가 V7 전용. V8.1은 옴니 없음 |
| 비율 | **2:3 (Portrait)** | 카드 UI가 aspect-[2/3]. 9:16 아님! |
| Weirdness / Variety | **둘 다 0** | 덱 일관성의 기본 |
| Personalize (P) | **끄기** | 개인화 프로필이 그림체를 틀어버림 |
| Style Reference | **확정 마스터 이미지** | 화풍 잠금. `--sw 600` |
| Omni Reference | **같은 마스터 이미지** | 물건/캐릭터 생김새 잠금. `--ow 300` |
| 파라미터 | `--ow 300 --sw 600 --no frame, border, text, letters, characters, watermark, person, face, eyes, tree, plant, stack` | 프롬프트 맨 끝, 영어만 |

**레퍼런스 원칙:**
- 스타일 마스터 = `이미지/만신타로/옥황_스타일마스터_클린.png` (**2026-07-12 확정.** 기존 스타일마스터에서 우하단 서명만 인페인팅으로 제거한 판 — 그림체는 기존과 동일. 서명 있는 원판·완성본을 sref로 쓰면 결과물마다 낙관/서명이 전염되며 `--no signature, stamp, seal`로는 못 막음)
- ⚠️ `신령카드/옥황상제_클린마스터.png`(완성본 유래)는 **sref로 쓰지 말 것** — 완성본은 광택·명암이 강해 그림체가 표류함 (실패 18). 신령패 옥황 얼굴이 필요할 때 **옴니**로만 사용
- 엽전 시리즈는 **확정 한 닢**이 마스터 (Style + Omni 동시 장착)
- **체인 금지**: 두 닢을 참조해 세 닢을 뽑는 식으로 직전 결과물을 갈아끼우면 복사본의 복사본이 되어 스타일이 표류함. **항상 같은 마스터 1장에서 출발**
- 신령 캐릭터도 동일: 옥황 마스터 고정. (물건 카드는 그 부류의 확정 1호를 옴니로 추가)

## 2. 프롬프트 공통 템플릿

```
Stylized card art for a Korean shamanic oracle deck, charming storybook illustration style.
[피사체 — EXACTLY N (숫자) 이중 표기 + 배치 묘사].
Seen at eye level, a warm beam of golden light falling from above, golden glow on the ground
beneath them, dark misty background with faint floating sparks.
The coins are INANIMATE OBJECTS - they have NO faces, NO eyes, NO expressions.  ← 사물 카드만
The coin surfaces are smooth and worn with NO characters, NO letters, NO markings engraved on them.
Rich deep color palette with gold-leaf accents. NO frame, NO border.
--ow 300 --sw 600 --no frame, border, text, letters, characters, watermark, person, face, eyes, tree, plant, stack
```

**간결성 원칙 (2026-07-14 확정 — 최우선):**
- **프롬프트가 길수록 개별 요소가 누락될 확률이 올라간다.** 방어 문장을 기계적으로 전부 쌓지 말 것.
- 카드별로 **핵심 리스크 1~2개만 골라 방어**하고 나머지는 뺀다 (예: 인물 카드에 INANIMATE 문장 불필요, 사물 카드에 피부톤 문장 불필요).
- `--no` 금지어도 그 카드에서 실제 나올 법한 것만 6~10개 이내로. 전 카드 공통은 `frame, border, watermark, signature` 정도.
- 문장 수 목표: **본문 5~6문장 이내.**

**우리 덱의 장면 문법 (모든 카드 공통 — 이 문장이 세트감의 핵심):**
> Seen at eye level, a warm beam of golden light falling from above, golden glow on the ground beneath them, dark misty background with faint floating sparks.

- 눈높이 시점 + 상단 광선 + 바닥 금빛 발광 + 어두운 안개 배경. 이 4요소가 빠지면 혼자 다른 카드가 됨.
- 신령 캐릭터용으로 바꿀 땐 fair porcelain skin tone, Korean facial features 추가 (당금애기 피부톤 이국적으로 나온 사례).

## 4. 도구 사용법 (리롤 전에 시도할 것)

1. **Vary Region (부분 수술)** — 이미지 → Edit → 문제 부위만 브러시 → 짧은 프롬프트. 마름모 구멍→"square hole", 가짜 글자→"smooth surface", 초과 엽전→"dark misty ground"로 삭제
2. **Vary Subtle** — 구도 좋은데 디테일 아쉬울 때
3. **Upscale Subtle/Creative** — 최종본 화질 상향
4. **리롤** — 장면 자체가 틀렸을 때만 (나무, 탑뷰, 배경 이탈)

판단 순서: "80점, 한 군데 아쉬움" → Vary Region / "60점" → Vary Subtle / "장면 틀림" → 리롤

## 5. 검수 체크리스트 (4장 중 고를 때, 장당 10초)

1. **개수 정확한가** (엽전패 1순위)
2. 테두리가 안 그려졌는가
3. 글자/뭉개진 한자 흔적 없는가
4. 얼굴(의인화) 없는가
5. 네모 구멍 온전한가 (마름모 변형 체크)
6. **장면 문법 4요소** — 눈높이 / 상단 광선 / 바닥 발광 / 안개 배경
7. 마스터와 같은 화풍인가 (선 굵기·색감·질감)
8. **100% 확대로 소품 전수 검사** — 타원형 소품(고추·엽전·돌·열매)의 눈·얼굴, 빈자리의 꼬마 인형·해골·생물 침입. 썸네일 판정 금지 (실패 22·23)

## 8. 사용량 관리 (과금 체계 2026-07-13 조사)

- 과금은 장수가 아니라 **GPU 시간제**: 1회 생성(4컷) ≈ 1 GPU분, 레퍼런스 걸면 증가. Fast 시간은 결제일(매월 8일) 리셋, **이월 없음**
- Basic 플랜($10): Fast 3h20m/월, **Relax 무제한 없음** (소진 시 그 달 생성 불가)
- 추가 Fast 시간 **$4/h** 수시 구매 가능, 이건 만료 없음 — 월말에 조금 모자라면 플랜 올리는 것보다 저렴
- 참고 실측: 엽전 6장 완성에 리롤 포함 약 40분~1시간 소모
- **양산 달에는 Standard($30, Fast 15h + Relax 무제한) 한시 업그레이드 권장** — 리롤·시행착오는 Relax로, Fast는 확정 컷 업스케일에만 (양산 정석 운영법)
- Basic/Standard 생성물은 **공개 갤러리 노출** (비공개는 Pro $60부터) — 물상/만신 IP 스타일이 미리 노출되는 점 인지
- 상업 이용은 유료 플랜이면 OK (연매출 $100만↑ 기업만 Pro 의무 — 해당 없음)
- V7이 V8.1보다 GPU 시간 더 소모
- 사용량 확인: midjourney.com/account → Usage Details
